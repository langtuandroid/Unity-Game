    Class ActionInstance.ActionConfig

    A configuration of the ActionInstance, each configuration has its own settings that affects the execution of the ActionInstance. This class should be subclassed inside subclasses of ActionInstance with name 'ActionInstance_Subclass_Name'Config. i.e CircleAttack which inherit from ActionInstance must have a nested class named CircleAttackConfig inherited from this class
